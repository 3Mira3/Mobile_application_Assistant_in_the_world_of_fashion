## Про проект

**«Помічник у світі моди»** — це інтелектуальний мобільний застосунок для Android, який допомагає користувачам створювати стильні та гармонійні образи з урахуванням:

- Реального гардеробу користувача
- Теорії кольорів
- Погодних умов
- Сезону та типу події
- Індивідуальних особливостей

Це **цифровий стиліст у кишені**, який поєднує освітній контент, персоналізовані рекомендації та сучасні технології штучного інтелекту.

### Основні модулі

- **Головна сторінка** + зручна навігація (Bottom Navigation)
- **Мій профіль** — налаштування користувача
- **Історія моди** — освітній розділ
- **Магія кольору** — інтерактивна колірна діагностика
- **Персоналізований підбір образів** — головна фішка застосунку
- **Наші стилісти** — довідник стилістів
- **Налаштування**

---

## Технологічний стек

- **Мова:** Kotlin
- **Архітектура:** MVVM + Clean Architecture елементи
- **UI:** Jetpack Compose + Material Design 3
- **Обробка зображень:** AndroidX Palette, Glide
- **Машинне навчання:** TensorFlow Lite (on-device)
- **Мережа:** Retrofit + wttr.in (погода), Pexels API
- **Локальне зберігання:** SharedPreferences, Room (план), in-memory кеш
- **Тестування:** JUnit + Espresso / Compose Testing

**Особливість:** Максимальна обробка даних **on-device** для конфіденційності та швидкості.
